# DrishtiAI — Vision Beyond Sight

Real-time traffic sign detection & voice guidance — empowering the visually impaired to navigate independently.

## Features
- YOLOv8 Real-time Detection
- PaddleOCR Text Recognition
- Multi-language Voice Output (English, Hindi, Marathi)
- WebSocket Streaming for Low Latency
